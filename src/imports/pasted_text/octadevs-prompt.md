# IMPORTANT TECHNICAL REQUIREMENTS (STRICT)

**THIS PROJECT MUST USE NEXT.JS ONLY.**

Do NOT generate React + Vite.

Do NOT generate a single-page React application.

Do NOT use Create React App.

Do NOT simulate pages using React state.

Generate a **production-ready Next.js project** using:

* Next.js (Latest Stable Version)
* App Router (Strictly)
* TypeScript
* Tailwind CSS
* Framer Motion
* Lucide React
* Next.js Image
* Next.js Link

Generate a clean folder structure.
